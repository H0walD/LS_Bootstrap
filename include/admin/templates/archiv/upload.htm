Datei

Eine Datei wird immer in das aktuelle Verzeichnis geladen, befinden Sie sich z.B. gerade in einem Ordner 'Movies/Fun' wird die Datei dort rein geladen.

Neue Ordner anlegen können Sie nur indem Sie im Ordner include/downs/downloads eine neue Ordner Struktur erstellen und jedem Ordner CHMOD Recht 777 geben. Sie können dort z.B. den Ordner Movies anlegen müssen diesem aber die nötigen Rechte geben. Sie müssen diese Ordner z.B. per FTP-Programm direkt auf Ihrem Webspace erstellen.

Die Spalte 'linked': Sie können der eben hochgeladenen Datei einen Download erstellen indem Sie auf 'neu' klicken. Die Zahlen in diesem Feld stellen bereits bestehende Downloads für diese Datei dar, Sie können diese Downloads ändern indem Sie auf die Zahl klicken.
